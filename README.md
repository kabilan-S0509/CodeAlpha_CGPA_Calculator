# CodeAlpha_CGPA_Calculator
# CodeAlpha C++ Internship Project
## Project: CGPA Calculator

A C++ console application that calculates CGPA from grades and credit hours.
It displays per-course grades, semester GPA, and overall CGPA.

### Features
- Handles multiple semesters
- Displays detailed table of course data
- Calculates weighted CGPA
- Gives feedback based on performance

### How to Run
```bash
g++ CGPA_Calculator.cpp -o CGPA
./CGPA   # (Windows: .\CGPA)
